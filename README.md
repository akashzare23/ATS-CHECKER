# ATS Resume Checker

## 🚀 Description
This project checks how well a resume matches job requirements using ATS (Applicant Tracking System) logic.

## 🔥 Features
- Upload resume
- Keyword matching
- Resume score
- Suggestions to improve resume

## 🛠 Tech Stack
- Python
- NLP
- Streamlit / Flask

## ▶️ How to Run
1. Install required libraries
2. Run the main file
3. Upload your resume
